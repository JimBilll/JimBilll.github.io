# JimBilll.github.io

This website is a basic demonstration of Jekyll working in GitHub. It contains a homepage, about page, blog link page and two blog entries.

The site's structure is:\n
    - **Homepage**\n
    - About Page\n
    - Blog\n
        - "Site Launched" Entry\n
        - "Second Post" Entry\n

The site has two different page layouts: Default - for main pages; Post - for blog posts.
